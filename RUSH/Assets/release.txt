- Ouvrir la scene "MainScene".
- Dans la scene activer le Level1
- Dupliquer les objects ou ajouter une action du dossier Prefabs/LevelActions (ou dans Prefabs/LevelElements pour les wall)
     - Attention les LevelActions on une boite englobante -> c'est celle-ci qu'il faut bouger et non les GameObjects asset à l'interieur
- Click sur Apply puis désactiver le Level1


Spécification par rapport au LevelActions : 
    -> Il faut renseigner les téléporteurs de déstination
    - Pour les LevelActionTeleportBlue -> il faut en poser deux sur la scène et pour les deux teleports posés sur la scene,
      drag & dropper dans le script Level Action Teleport des GameObject, l'autre teleport dans "Destination Teleport"
